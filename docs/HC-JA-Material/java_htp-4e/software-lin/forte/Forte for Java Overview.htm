Introducing Forte[tm] for Java[tm], release 2.0 software--the powerful, extensible, scalable integrated development environment (IDE) for Java technology developers.

Sun's Forte for Java software is the only development tool needed for building Java applications on any platform. It is fully modular and easily extensible because it's completely open and based on the NetBeans[tm] Tools Platform. The Forte for Java IDE enables you to easily and quickly create Internet services and solutions with 100% Pure Java[tm] code on the Windows, Linux, or Solaris[tm] Operating Environments.

This powerful IDE simplifies the tasks you perform most often: compiling, building, browsing, editing, and debugging. Its outstanding GUI builder and JavaBean[tm] assembly tools make it easy to get and stay productive, and it even includes a JPDA-compatible debugger for distributed services and applications. Depending on your development needs, you can choose from two editions of the Forte for Java product:

  • The Community Edition product is offered at NO CHARGE and includes a complete and highly-integrated set of tools-- including a Web browser and a Web server. With this edition, any developer can build stand-alone applications, applets, JavaBeans, and Java clients.

  • The Internet Edition product includes all the functionality in the Community Edition plus support for teams of developers building database-aware Web applications. This edition includes integration with Tomcat (a Java Server Pages 1.1/Servlets 2.2 reference implementation), and it expands on the functionalities of the Web browser and Web server in the Community Edition. Its plug-in components are ideal for speeding your dynamic Web content development.

Free Yourself From Vendor Lock-In!

Because Forte for Java software is based on the NetBeans Tools Platform, you're not locked into a static set of development tools. NetBeans is an open source, standards-based, modular, integrated development environment (IDE), written in the Java programming language. NetBeans participants contribute and share their source code with the community, support community discussions, and much more.

"I'm a real fan of NetBeans. It's easily my favorite IDE. My favorite features are that it works everywhere I want to work (Linux, Solaris, and Windows) and with its Open APIs, it's incredibly flexible."

James Gosling
Sun Fellow & Vice President
Sun Microsystems

The Forte for Java software enables you to build Java applications on any platform because it has an open modular architecture. Even the Java language support is implemented in a pluggable module!

Opportunities for Partners and Developers

The Forte for Java product strategy provides independent software vendors (ISVs), integrators, software houses, and end users with attractive business and technology opportunities. To start-up your business, you can access the Forte for Java developer community, and you can sell compatible products and services in the on-line marketplaces. In addition to customer access, Sun gives you technology that speeds your time to market. The Forte for Java platform is built on flexible APIs that facilitate creation of plug-in modules without having to modify our binaries. The architecture makes it straightforward to build modules and components that seamlessly integrate with the IDE.

Sun has exciting partner programs and a growing list of partners, who offer a wide variety of extension products and services.

Fast, Productive Development Environment

The Community Edition provides integrated, easy-to-use features to help you get started fast in programming in the Java language. Its seamless interface provides developer tools, such as an application browser, Java language source editor, debugger, compiler, online Help, and much more. Development wizards enable you to quickly and easily create and deploy JAR files, generate source documentation, and more. An advanced debugger helps you quickly find bugs--even in distributed and/or multiplatform applications.

New Features for Internet-Savvy Services and Solutions

The Internet Edition builds on the Forte for Java, Community Edition, and is ideal for dynamic Web content services and solutions. It includes tools to help you quickly develop JavaServer Pages[tm] (JSPs) and Servlets. Its support for JDO-based Transparent Persistence, XML, CORBA, RMI, and JDBC code development enable you to easily create database-aware services and solutions.

Parallel Development Support

The Internet Edition also includes Forte TeamWare configuration management software that provides your team with a simple but powerful solution for parallel development. Forte TeamWare includes version control, change control, source integration, and release management features that will improve your team's productivity as well as your product's quality, and gets your services and solutions to market faster.
