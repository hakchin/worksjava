JMF 2.1.1-FCS

Requirements

  • Blackdown or Sun Java2 Standard Edition SDK or Runtime Environment v1.3.0 or higher for Linux/i386

Installation

JMF 2.1.1 is currently available in three different formats. You only have to install one of the following packages:
  • Standard package: jmf-2.1.1-fcs-linux-i386.tar.bz2
    Download jmf-2.1.1-fcs-linux-i386.tar.bz2 from one of our mirrors.

    Installation: 
    % cd <targetdirectory>
% bzip2 -dc <downloadpath>/jmf-2.1.1-fcs-linux-i386.tar.bz2 | tar xvf -
% export PATH=<targetdirectory>/jmf-2.1.1/bin:$PATH
% export CLASSPATH=<targetdirectory>/jmf-2.1.1/lib/jmf.jar:<targetdirectory>/jmf-2.1.1/lib/sound.jar:<targetdirectory>/jmf-2.1.1/lib/mediaplayer.jar:<targetdirectory>/jmf-2.1.1/lib/multiplayer.jar:<targetdirectory>/jmf-2.1.1/lib:$CLASSPATH
% export LD_LIBRARY_PATH=<targetdirectory>/jmf-2.1.1/lib:$LD_LIBRARY_PATH

  • Java2 Extension package: jmf-for-java2-2.1.1-fcs-linux-i386.tar.bz2
    Download jmf-for-java2-2.1.1-fcs-linux-i386.tar.bz2 from one of our mirrors.

    • Installation with a J2SDK 1.3: 
      % cd <j2sdk1.3.0 installation>/jre
% bzip2 -dc <downloadpath>/jmf-for-java2-2.1.1-fcs-linux-i386.tar.bz2 | tar xvf -
% export PATH=<j2sdk1.3.0 installation>/jre/bin:$PATH
    • Installation with a J2RE 1.3: 
      % cd <j2re1.3.0 installation>
% bzip2 -dc <downloadpath>/jmf-for-java2-2.1.1-fcs-linux-i386.tar.bz2 | tar xvf-

  • Debian package: jmf_2.1.1-1_i386.deb
    Select one of our mirrors and add a line of the form
    deb ftp://<mirror>/<path to java dir>/debian <dist> non-free

    to /etc/apt/source.list. <dist> should be potato or woody.
    E.g., if you want to use the woody tree on ftp://ftp.tux.org/ you would use

    deb ftp://ftp.tux.org/java/debian woody non-free

    Then install the package jmf using the standard Debian tools. No further setup is required.
