Getting Started

To get started using this CD, you need to install the software packages. Before you begin, however, make sure your computer meets the following System Requirements.

System Requirements

Operating System

In order to use everything on this CD, you need to be running Windows 95, Windows 98, Windows NT or Windows 2000.

Processor

To use everything, at a minimum you need to have a 166 MHz (minimum) or faster Pentium / Compatible processor. Forte for Java requires a minimum 350 MHz processor.

Graphics Display Card / Sound Card

A standard graphics display card is adequate. For audio playback using the Java Media Framework, if your Windows machine does not have built-in audio support, you need a sound card.

RAM

To use everything under Microsoft Windows you need a minimum of 128 MB of RAM, but 256 MB is recommended.

Disk Space

To install everything on the CD, it is recommended that you have at least 800 MB of hard disk space available.

Some materials on this CD may require the use of a Zip archiving utility. If you do not have one, you can download a free evaluation version of WinZip at http://www.winzip.com

Installing the Software

Now that you have verified that your system meets the minimum requirements for installing the software, you must read the license agreements. Please read the entire text of each of the license agreements and make sure you agree with them. Now you are ready to install. For instructions on how to install each of the software packages, click here.

Examples

Once you have installed all the software, you are ready to begin working through the exercises in the book. The examples directory can be viewed by clicking directly on the Examples on the startup screen or by clicking here.

This will bring you to a window that shows the examples directories of the CD. They are arranged by chapter number and contain the example code for each particular chapter. You can copy them to your hard drive and modify them as you work your way through the book.
