Java™ Media Framework
for Linux Installation

Read the following documentation before you install Java™ Media Framework from the CD.

Have you reviewed the Java™ Media Framework License Agreement?

Requirements

Blackdown or Sun Java2 Standard Edition SDK or Runtime Environment v1.3.0 or higher for Linux/i386

Installation

You only have to install one of the following packages, both files are located in the software-lin\framework directory on the root level of this CD-ROM.

  • Standard package: jmf-2.1.1-fcs-linux-i386.tar.bz2

    • Download jmf-2.1.1-fcs-linux-i386.tar.bz2 from the CD to your hard drive using your favorite graphical or command line tool, then follow the instructions below.

    Installation:

    % cd <targetdirectory>
    % bzip2 -dc <downloadpath>/jmf-2.1.1-fcs-linux-i386.tar.bz2 | tar xvf -
    % export PATH=<targetdirectory>/jmf-2.1.1/bin:$PATH
    % export CLASSPATH=<targetdirectory>/jmf-2.1.1/lib/jmf.jar:<targetdirectory>/jmf-2.1.1/lib/sound.jar:<targetdirectory>/jmf-2.1.1/lib/mediaplayer.jar:<targetdirectory>/jmf-2.1.1/lib/multiplayer.jar:<targetdirectory>/jmf-2.1.1/lib:$CLASSPATH
    % export LD_LIBRARY_PATH=<targetdirectory>/jmf-2.1.1/lib:$LD_LIBRARY_PATH

  • Java2 Extension package: jmf-for-java2-2.1.1-fcs-linux-i386.tar.bz2

    • Download jmf-for-java2-2.1.1-fcs-linux-i386.tar.bz2 from the CD to your hard drive using your favorite graphical or command line tool, then follow the instructions below.

    • Installation with a J2SDK 1.3:
      • % cd <j2sdk1.3.0 installation>/jre
      % bzip2 -dc <downloadpath>/jmf-for-java2-2.1.1-fcs-linux-i386.tar.bz2 | tar xvf -
      % export PATH=<j2sdk1.3.0 installation>/jre/bin:$PATH
    • Installation with a J2RE 1.3:
      • % cd <j2re1.3.0 installation>
      % bzip2 -dc <downloadpath>/jmf-for-java2-2.1.1-fcs-linux-i386.tar.bz2 | tar xvf-

Data Integrity Error

If you are extracting the archive using the command prompt and receive an error that states 'Data integrity error when decompressing,' please do one of the following:

  1. Check the directory location where you were expanding the archive. The expanded directory may be intact.

  2. Use a GUI archive utility such as 'ark' to expand the archive.
