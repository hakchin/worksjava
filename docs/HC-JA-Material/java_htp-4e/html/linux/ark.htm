Ark is a program for managing and quickly extracting archives. It works closely with KFM and various other parts of the KDE environment.

Current features include: a quick list of archives in a user defined directory, quick extraction of archives to any directory, and adding files via drag and drop from KFM. Ark supports .ar, .lha, .tar.gz, .tar.bz2, .tar.lzo and .zip archive formats.

Ark is part of the kdeutils package.
